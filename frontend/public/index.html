Your Micro App chatbot is not here.

You need to load a shell Container.

You can either use Webpack 5 module Federation or Single SPA. Both are supported.

For Webpack 5 module Federation, you need to add the following configuration to your webpack.config:

plugins: [
new ModuleFederationPlugin({
...moduleFedConfigs,
remotes: {
'chatbot': './remoteEntry.js'
},
})
],
